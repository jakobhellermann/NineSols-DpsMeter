# DPS Meter

- Statistics panel for DPS by damage source
- damage popup numbers
- timing information on inaccurate parry

## Configuration
- Show damage numbers - Whether the popups should be didsplayed
- Reset Mode
    - Manual Resets: DPS stats are only reset when you hit the keyboard shortcut
    - Lock To First Enemy: After hitting an enemy, only damage to that enemy will be tracked
    - Reset On Enemy Change = Switching the enemy automatically resets the DPS tracking
- Keyboard shortcuts for DPS Tracking Pause & Reset
